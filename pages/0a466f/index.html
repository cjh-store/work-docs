git发布tag

  1. 将分支切换为master分支

    git checkout master

  2. 合并develop分支代码

    git merge develop

  3. 运行tag命令

    yarn git:tag
如果命令失败 先运行yarn安装依赖

  4. 选择更新的版本号

  5. 完成
